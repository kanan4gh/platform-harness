# ドキュメントレビュー（Kiroアダプタ）

**手順の正は `docs/procedures/review-docs.md` にある。必ず全文を読み、5観点・文書種別別チェック・レポート形式に従うこと。**

## Kiro固有の割当

- 著者と独立した文脈での実行には`.kiro/agents/doc-reviewer.md`をサブエージェントとして使う
- カスタムサブエージェントが利用できない場合は、Kiroの既定サブエージェントへ対象ファイルと中立手順を明示して委譲する
- レビュー依頼は読み取り専用で行い、ユーザーが修正を求めない限りファイルを変更しない
